Microsoft NERD Center

One Memorial Drive
Cambridge, Massachusetts 02142

Wednesday, May 28, 2014 9:00 AM - 5:00 PM

 

Join Unity & Microsoft for an awesome, one-day workshop where you’ll learn how to build a game with the new 2D toolset and port it to Windows Phone or Windows Store. This FREE event is designed to accommodate both Unity beginners and Unity experts - and we’ll keep everyone fueled with complimentary food and beverages. Attendees will be eligible to win some great prizes, including software licenses, the latest Windows phone devices, and a special grand prize. If you want to see your app on the Windows platforms, this is the event to attend!

Agenda for the Day

Session 1

Publishing Games on Windows

Learn the processes and requirements for publishing your game to both the Windows Store and the Windows Phone Store. Why? Windows Phone is outselling the iPhone in 24 markets, and Windows 8 has surpassed 150 million licenses. It’s time to push your games out in a marketplace that’s not overly saturated - and we’ll show you how!

Unity skill level: beginner & expert

Session 2

Introduction to Unity

Microsoft and Unity experts will guide you through the toolset and teach you how to build a 2D game and export it to Windows and Windows Phone. You could walk away with enough fundamental knowledge to bring your very own game ideas to life!

Unity skill level: beginner

Unity Porting Lab

In this hands-on lab, Microsoft and Unity pros will show you how to port your Unity games to Windows and Windows Phone. Bring your computer and your pre-existing game so you can make the most of this session. It’s time to sell you work on the Windows platform!

Unity skill level: expert

Session 3

Demos, Voting, & Prizes

Beginners and experts will gather to showcase their ported games in a series of fast-paced, 2-minute demos. The audience will vote for the best apps and we’ll give away great prizes, including software licenses, the latest Windows Phone devices and a special grand prize.

What you need to bring with you:

A PC running Windows 8 or 8.1 with Visual Studio 2012 or 2013 and Unity 4.3.3+ installed. Building for Windows Phone also requires the Windows Phone 8 SDK.

-OR-

For Mac users, we will provide Windows QuickStart kits (which will run Windows 8 pro on your Mac OSX via Parallels Desktop 9).

*Windows and Windows Phone devices will be available for testing.

*Breakfast and lunch will be provided.
