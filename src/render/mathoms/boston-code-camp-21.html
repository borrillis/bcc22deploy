Microsoft New England R&D (NERD), One Memorial Drive, Cambridge, MA
Session Submissions for Boston Code Camp 21
open on Wednesday, April 16, 2014

This spring, the Greater Boston development community will unite again for the 21st outstanding community-based software (un-)conference like no other.

Code Camp brings regional software development professionals together for the opportunity to immerse themselves in seminars, presentations, group exploration, and networking. Participants will be able to engage in their preferred technology, as well as to sample other options, with a focus on extending information exchange and enhancing the cross pollination of ideas. Mark your calendar! Boston Code Camp 21 is scheduled for Saturday, June 21, 2014 at Microsoft New England R&D (NERD), One Memorial Drive, Cambridge, MA.

NERD is at One Memorial Drive, Cambridge, MA 02142

Admission to this event is free! All costs are covered by sponsors and individual contributors.
