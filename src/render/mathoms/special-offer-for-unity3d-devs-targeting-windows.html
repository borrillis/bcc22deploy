Building Games for Windows Using Unity3d?


We've got an incentive offer for you. Unity developers can receive complimentary store registrations, Windows 8 licenses, and phone and tablet devices to work with. There's also promotional packages and technical support offerings as well. Sign up here http://unity3d.com/pages/windows/offer  . Also read the terms and conditions before digging in.

As always, if you're a Unity3D developer from the New England Area, you can always come to a Studio Time session to get more assistance. If you release a game or app to either the Windows Store or the Windows Phone Store let me know and I'll see what I can do to help drive excitement for your games and apps.
